# 간단한 게임 만들기

[요구사항]
1. 게임이 시작하면 '게이머의 이름을 입력하세요. 단 28자 이내'
- 만약 입력을 안 하고 엔터를 치면 -> 경고메시지 -> 다시 입력을 받는다
- 만약 28자 초과(>28) 입력하면 -> 경고메시지 -> 다시 입력을 받는다
- 정상적으로 입력을 받으면 1단계 끝

2. 게임 프롬프트 출력
===========================
숫자 맞추기 게임(1~99)
v 1.0.0
이름
===========================
- 중앙정렬 필요 X
- 입력받은 이름을 포함한 프롬프트 출력
