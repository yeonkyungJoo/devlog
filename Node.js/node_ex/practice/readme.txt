* Ajax
- js 기반으로 백그라운드에서 비동기 통신을 담당
- 특징
    - 화면 껌벅임 X
    - 동일 도메인에서만 사용 가능(동일 사이트에서만 가능)
    - 다루는 데이터는 html, xml(1999), text, json(2007~2009 후반) 등 텍스트 포맷 가능
    1) xml
    - 플랫폼과 언어에 독립적인 데이터 포맷으로 최초로 등장한 형식
    - json의 등장으로 주류에서 밀려났고, 여행사, 항공사, 숙박 등의 계열에서는 아직 사용
    - 환경 구성 등 포맷으로 사용
    - html의 엄격한 규격의 영향을 지속적으로 미치고 있다
    2) json
    - 전자정부프레임워크(spring 주력)에서 주된 데이터 포맷으로 사용
    - JavaScript Object Notation
    - xml 대비 같은 정보를 저용량으로 표현하게 되어서 각광
    - 2010년 이후로는 대부분의 http기반 통신 base의 데이터 형식을 주로 사용
    
* jQuery
- 자바스크립트의 프레임워크
- js를 보다 손쉽게 개발하기 위해 사용
- CSS selector, 이벤트 처리, 통신(ajax), DOM 조작(화면 조작)
